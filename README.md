# Overview
- グーチョキパーのどれかの画像と、「以下に勝て」、「以下に負けろ」、「以下に引き分けろ」のテキストが同時に表示される。
- それらの案内にしたがってカメラからジェスチャーを入力する。

# Requirement
- Python3.8 or newer
- Flask
- mediapipe
- OpenCV
- Numpy

# Deploy 
```
$ /path/to/your/janken-exercise: pip3 install -r requirements.txt
```

# Run
```
$ /path/to/your/janken-exercise: python3 app.py
```

http://0.0.0.0:5000/
